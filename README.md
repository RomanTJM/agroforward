# Агрофорвард — Корпоративный сайт

Корпоративный сайт компании **ООО «Агрофорвард»** — специализированного оператора поставок мясного сырья для переработчиков и розничных сетей России.

---

## Технологии

Бибилиотека - React 19 
Роутинг - React Router DOM 7 
Анимации - Framer Motion 12 
Иконки - Lucide React 
Графики - Recharts 3 
Формы - React Hook Form 7 
PDF - jsPDF 4 
Стили - Vanilla CSS + CSS Custom Properties
Сборка - Vite 8  
Линтер - ESLint 9 

---

## Запуск

```bash
# Копирование репозитория
git clone https://github.com/RomanTJM/agroforward.git
cd agroforward

# Установка зависимостей
npm install

# Режим разработки
npm run dev

# Сборка для продакшена
npm run build

# Предпросмотр сборки
npm run preview
```

Приложение запускается на `http://localhost:5173`.

## Архитектурные решения

**Компонент `Carousel`** (`src/components/common/Carousel.jsx`) — переиспользуемый бесконечный карусель, используется на трёх секциях сайта. Вся логика позиционирования реализована через прямое DOM-управление (`ref.style.transform`) без React-рендеров, что обеспечивает нулевую задержку при drag-взаимодействии. Поддерживает инерцию (velocity-based inertia) и адаптируется под мобильные устройства (3 карточки → 1).

**CSS Custom Properties** — дизайн-токены (цвета, радиусы, тени, градиенты) объявлены в `src/index.css` и используются во всех компонентах. Нет UI-библиотек — всё написано вручную.

**Данные вынесены в `constants/`** — все текстовые и конфигурационные данные страниц хранятся отдельно от компонентов, что упрощает редактирование контента.

**Изображения в `src/assets/`** — все медиафайлы импортируются через Vite, что позволяет бандлеру оптимизировать и хешировать их при сборке.
